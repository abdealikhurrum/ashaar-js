Justification modes

10. CSS-native justification — ashaar--justify class

Adds text-align: justify; text-justify: auto — browser uses kashida for RTL Arabic. Works in Chrome/Edge; Firefox falls back to word spacing.

قِفَا نَبْكِ مِنْ ذِكْرَى حَبِيبٍ وَمَنْزِلِ \ بِسِقْطِ اللِّوَى بَيْنَ الدَّخُولِ فَحَوْمَلِ بِعَيْنَيَّ هَلَّا بَكَّيَا أَوْ تَبَكَّيَا \ عَلَى النَّأْيِ مِنْ سُعْدَى وَمَا قَدْ تَوَلَّيَا

11. JS kashida justification — justify: true

Explicit tatweel (U+0640) insertion via DOM measurement + binary search. Each column filled to its exact available width. Responsive via ResizeObserver.

أَلَا لَيْتَ شِعْرِي هَلْ أَبِيتَنَّ لَيْلَةً \ بِوَادٍ وَحَوْلِي إِذْخِرٌ وَجَلِيلُ وَهَلْ أَرِدَنَّ يَوْمًا مِيَاهَ مَجَنَّةٍ \ وَهَلْ يَبْدُوَنَّ لِي شَامَةٌ وَطَفِيلُ
